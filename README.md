## 简介

- gitbook 页面访问统计插件

## 使用

#### book.json


```
{
  "plugins": ["site-count"],
  "pluginsConfig": {
    "site-count": {
      "host": "https://www.answera.top"
    }
  }
}
```

#### install

```sh
$ gitbook install
```
